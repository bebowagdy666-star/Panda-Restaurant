Panda Restaurant

Note: This project was built using a light copy from a design reference for learning purposes.

Project Overview

Panda Restaurant is a modern restaurant website built using HTML and CSS. The website features a clean, dark-themed design with vibrant gradients and smooth animations. It is fully responsive and works on both desktop and mobile devices.

Sections
1. Header / Navigation

Sticky header that stays at the top of the page.

Contains the restaurant name, tagline, and navigation links.

Smooth hover effects on navigation links with gradient underline animation.

2. Home Section

Large welcoming heading with the restaurant name.

Two call-to-action buttons: “Explore Menu” and “Contact Us”.

Buttons have hover animations with translate, scale, rotate, and gradient effects.

3. About Section

Cards highlighting key information about the restaurant.

Each card has hover effects including shadow, lift, and smooth transitions.

Use of semi-transparent backgrounds with blur for a modern glassmorphism look.

4. Menu Section

Grid layout displaying different dishes with descriptions.

Cards have hover effects with gradients and slight lift animation.

Text uses gradient headings for a vibrant effect.

5. Gallery Section

Grid layout showing restaurant dishes or interior images.

Images have hover animations: scale up, slight rotation, and shadow.

6. Contact Form

Simple contact form with input fields for name, email, and message.

Rounded corners with focus effect glow.

Submit button has gradient background with hover animation.

7. Footer

Simple footer with centered text and light gray color.

Contains copyright or additional info.
